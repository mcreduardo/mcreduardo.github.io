SilageSnap

SilageSnap is an app avilable for both iOS and Android. It’s capable to assess particle size distribution of corn kernels in corn silage dairy feed using image analysis. It provides the agricultural community an easy method to estimate the kernel size distribution on the field. I was responsible for the development of the image processing code in C++11 using OpenCV.

Special thanks to the Midwest Forage Association and the Baldwin Wisconsin Idea Endowment for making this research and development possible!

UnBeatables (2017)

UnBeatables is a technology and social project from the University of Brasilia. It is affiliated to the Automation and Robotics Laboratory (LARA). The project focuses on the programing of humanoid robots for Humanoid Robots Soccer competitions and social programs. The team gets involved with the community by attending events and going to schools, where people get to interact with the autonomous robots. The team has won many prises, including first place in the international competition RoboCup in the category Standard Platform League (Drop-in Only) multiple times.

I worked in the behavioral team, responsible to implement the autonomous behavior of the robot using state macines through an object oriented implementation in C++11. The team uses the plataform NAO from Aldebran Robotics.

Draco Volans (2014-2015)

Draco Volans is a competition team founded in 2003 at Universidade de Brasília with the purpose to copete at SAE Aero Design Regular Class. These competitions generally have the objective of the design of fixed-wings unmaned aerial vehicles capable to carry as much load as possible given dimension constraints. The team is renowned by excellency in the theorical design and, even though it doesn’t have financial support as big as some other teams, it is currently between the five top teamns in South America.

I was part of the structural design team, with the specific task of writing code for structural design simulation and simulate models using the software Ansys.
