SilageSnap

Silage Snap Application

SilageSnap is an app avilable for both iOS and Android. It is capable to assess particle size distribution of corn kernels in corn silage dairy feed using image analysis. It provides to the agricultural community an easy method to silage quality in the field. I was responsible for the development of the image processing code using C++ and OpenCV, and for the iOS app basic infrastructure using Swift.

Special thanks to the Midwest Forage Association and the Baldwin Wisconsin Idea Endowment for making this research and development possible!

Multiple Linear Regression Application

Photo

Multiple Linear Regression Application with User Interface ( R )

This is an application made in R that allows the user to do Multiple Linear Regression without any knowledge of programming. The app has a graphical interface made with the package Shiny. The regression is made by simply uploading a txt file with the data and defining a regression formula with any variables contained in that file.

The application provides:

  • Suuuummary with all fitted coeffients and respective t statistics;
  • R2, R2adj and BIC of model;
  • ANOVA table with f statistics;
  • Residual plots: res vs fitted, QQ plot, leverage, Cook’s distance;
  • Multicolinearity analysis: VIF, correlation, and correlation plots;
  • Model selection with analylis of nested models by R2, R2adj and BIC.

Maple Syrup Sugar Level Prediction

Video

Maple Syrup Sugar Level Prediction ( R )

Contact me for the full article.

Abstract:

Sap is collected from trees for the production of Maple Syrup. The higher the concentration of sugar in the sap, the more time and money is saved by a producer. This study has the objective to develop a model to predict the sugar level in the sap given a few predictor variables. A first look at the data showed that the variables Sun and Days appear to have a clear relation with the sugar level. Testing different models, the power transformation Sugar2 resulted in a better full model that met the model assumptions better. Using exhaustive search, the model without the variables Growth and Temp met all its assumptions and resulted in maximum BIC and adjusted R2 of 0.837. Therefore, it was chosen as best model. The predictor variable Sun is dominant in all good models found. The best model without it has adjusted R2 of 0.466. All assumptions of the final model were met. To maximize syrup production based on the model obtained, sap should be collected from trees with bigger DBH and more sun exposure as close to the first tapping of the tree as possible.

Factorial Analysis of Allergy Treatments

Video

Factorial Analysis of Allergy Treatments ( R )

Contact me for the full article.

Abstract:

Two new allergy treatments are being tested. The study goal is to analyze the data and extract important information about the treatments combinations. The problem was modeled as a factorial treatment structure. Initial observations by an interaction plot indicated that both treatments reduce symptoms’ score and that there is interaction between them. The model was fitted and its assumptions of normality, constant variance, and linearity were met, but there might be a problem with independence depending on the study’s target population. Formal tests showed evidence of main effects for both treatments and antagonistic interaction. The oral treatment seems to be more important since it resulted in bigger main effect values. Fisher’s LSD was used to compare the means corresponding to all combinations. It showed no evidence that the best five groups (the ones with lowest symptoms’ score) have different symptoms’ relief between each other. Between these five groups, no nasal therapy and oral therapy with dose of 10 mg was chosen as the best combination since it is the simplest.

Analog Gopinath Motion Observer for Velocity Self-Sensing

Video

Analog Gopinath Motion Observer for Velocity Self-Sensing

Contact me for the full report.

This was the final project for the Automatic Controls class taught by Dr. Robert Lorenz in Spring of 2018 at the University of Wisconsin-Madison.

The main goal of this project was to implement a Gopinath observer and evaluate the performance of the observer by analytical and experimental methods.

The limitations in designing and implementing the observer were studied by using the estimation accuracy metric and dynamic stiffness. The effect of parameter estimation errors on estimation accuracy of observer was determined along with investigation of the state feedback motion controller performance with the observer estimated velocity feedback.

UnBeatables (Universidade de Brasília)

UnBeatables is a social and research project at Universidade de Brasília affiliated to the Automation and Robotics Laboratory (LARA). The project focuses on the programing of humanoid robots for Humanoid Robots Soccer competitions and social programs. The team gets frequently involved with the community by attending social events and going to schools, where people get to interact with the autonomous robots. The team has won many prises, including first place in the international competition RoboCup in the category Standard Platform League (Drop-in Only) multiple times.

The team uses the plataform NAO from Aldebran Robotics, and has its own software infrastructure, responsible for tasks such as vision and controls. I worked in the behavioral team, responsible to implement the autonomous behavior of the robot and integrate all the other software modules (controls, vision, mapping, ...).

Draco Volans (Universidade de Brasília)

Draco Volans is a competition team founded in 2003 at Universidade de Brasília with the purpose to compete at SAE Aero Design Regular Class. These competitions generally have the objective of the design of fixed-wings unmaned aerial vehicles capable to carry as much load as possible given dimension constraints. The team is renowned by excellency in the theorical design and is frequently between the top ten teamns in South America.

I was part of the structural design team. I wrote code in C and Matlab for optimizing the structural design for minimum weight, and automated structural simulations using the software Ansys APDL.
