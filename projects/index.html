SilageSnap

SilageSnap is an app avilable for both iOS and Android. It’s capable to assess particle size distribution of corn kernels in corn silage dairy feed using image analysis. It provides the agricultural community an easy method to estimate the kernel size distribution on the field. I was responsible for the development of the image processing code in C++11 using OpenCV.

Special thanks to the Midwest Forage Association and the Baldwin Wisconsin Idea Endowment for making this research and development possible!

Multiple Linear Regression Application with User Interface (Shiny R)

This is an application I made in R that allows the user to do Multiple Linear Regression without any knowledge of programming. The app has a graphical interface made with the package shiny. The regression is made by simply uploading a txt file with the data and defining a regression formula with any variables contained in that file.

The application provides:

  • Summary with all fitted coeffients and respective t statistics;
  • R2, R2adj and BIC of model;
  • ANOVA table with f statistics;
  • Residual plots: res vs fitted, QQ plot, leverage, Cook’s distance;
  • Multicolinearity analysis: VIF, correlation, and correlation plots;
  • Model selection with analylis of nested models by R2, R2adj and BIC.

Analog Gopinath Motion Observer for Velocity Self-Sensing

This was the final project for the Automatic Controls class taught by Dr. Robert Lorenz in Spring of 2018 at the University of Wisconsin-Madison.

The main goal of this project was to implement a Gopinath observer and evaluate the performance of the observer by analytical and experimental methods.

The limitations in designing and implementing the observer were studied by using the estimation accuracy metric and dynamic stiffness. The effect of parameter estimation errors on estimation accuracy of observer was determined along with investigation of the state feedback motion controller performance with the observer estimated velocity feedback.

Project report: GopinathObs.pdf (.doc)

UnBeatables (2017)

UnBeatables is a social and research project at Universidade de Brasília affiliated to the Automation and Robotics Laboratory (LARA). The project focuses on the programing of humanoid robots for Humanoid Robots Soccer competitions and social programs. The team gets frequently involved with the community by attending social events and going to schools, where people get to interact with the autonomous robots. The team has won many prises, including first place in the international competition RoboCup in the category Standard Platform League (Drop-in Only) multiple times.

I worked in the behavioral team, responsible to implement the autonomous behavior of the robot using state macines through an object oriented implementation in C++11. The team uses the plataform NAO from Aldebran Robotics.

Draco Volans (2014-2015)

Draco Volans is a competition team founded in 2003 at Universidade de Brasília with the purpose to compete at SAE Aero Design Regular Class. These competitions generally have the objective of the design of fixed-wings unmaned aerial vehicles capable to carry as much load as possible given dimension constraints. The team is renowned by excellency in the theorical design and, even though it doesn’t have financial support as big as some other teams, it is currently between the five top teamns in South America.

I was part of the structural design team, with the specific task of writing code for structural design simulation in C and Matlab and simulating models using the software Ansys.
